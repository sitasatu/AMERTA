AMERTA — MCDA for Retention Ponds Site Suitability

A QGIS plugin to identify and prioritize potential retention ponds locations using Multi-Criteria Decision Analysis (MCDA).

QGIS ≥ 3.40 (LTR OK) License: GPL-2.0+ Windows / Linux / macOS

Key features

Rainfall analytics (CHIRPS)

Prepare annual rainfall (sum/mean) and variability via PCA (eigenvector) for MCDA scoring.

Gridded factors

Create analysis grids and populate per-cell factors: rainfall, TWI, soil type (JTNH), and land cover (PL).

MCDA scoring & combine

Compute factor scores (e.g., S_Rain, S_TWI, S_PL, S_JTNH) and aggregate to suitability.

Utilities

Standardize classes, clip to AOI, and generate clean, reproducible outputs with key fields.

Compatibility

  • QGIS: 3.40+ (LTR supported)
  • OS: Windows, Linux, macOS (tested on recent versions)
  • Dependencies: Uses standard QGIS/Processing; no external Python packages required for core modules.

Installation

Option A — From a release ZIP (recommended)

  1. Download the latest .zip from the repository’s Releases page.
  2. In QGIS: Plugins → Manage and Install… → Install from ZIP → select the file → Install.

Option B — From source

git clone https://github.com/sitasatu/AMERTA.git

#  • macOS:   ~/Library/Application Support/QGIS/QGIS3/profiles/default/python/plugins/
# Restart QGIS and enable the plugin via Plugins → Manage and Install…

Quickstart

  1. Enable the plugin: Plugins → Manage and Install… → search “AMERTA”.
  2. Open Processing Toolbox → group AMERTA.
  3. Typical flow:
    1. Prepare rainfall layers: CHIRPS – Annual (Sum/Mean) / PCA Variability.
    2. Build grids & factors: Grid Rainfall, Grid TWI, Grid Soil Type (JTNH), Grid Land Cover (PL).
    3. Combine factors with MCDA to get overall suitability (e.g., S_Total).
  4. Result: a grid layer with factor fields and a suitability class for retention ponds siting.

Modules overview

  • CHIRPS – Annual Rainfall (Sum / Mean / PCA Variability): prepares rainfall magnitude and variability drivers.
  • Grid Rainfall: adds mrain, eigen, scores (sa, sv), and S_Rain.
  • Grid TWI: classifies TWI and computes S_TWI.
  • Grid Soil Type (JTNH): assigns dominant soil type per cell and S_JTNH.
  • Grid Land Cover (PL): assigns dominant land cover per cell and S_PL.
  • MCDA Combine: weights/aggregates factor scores into S_Total and suitability classes.

Inputs & outputs

Inputs

  • AOI polygon (administrative boundary, watershed, etc.).
  • Raster layers: CHIRPS rainfall (mm/yr), PCA eigenvector (PC1), TWI.
  • Vector layers: standardized Soil Type (JTNH), standardized Land Cover (PL).

Outputs (typical fields)

  • Rainfall: mrain, eigen, sa, sv, S_Rain
  • TWI: TWI_class, S_TWI
  • Soil: JTNH, S_JTNH
  • Land Cover: PL, S_PL
  • MCDA: S_Total (0–100) + suitability class
